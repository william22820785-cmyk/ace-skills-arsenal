# MOVA Skills 速查卡 (Quick Reference)

> 常用Skills一键查找

---

## 🚀 日常高频操作

| 场景 | 命令 | Skill |
|------|------|--------|
| 发小红书 | `social-poster` | 社媒发布 |
| 查财经新闻 | `alphaear-news` | 财经热点 |
| 爬网页 | `crawl <url>` | 网页爬取 |
| 写文档 | `feishu-doc` | 飞书文档 |

---

## 🔥 营销必备

```bash
# 社媒发布
clawhub install social-poster

# 内容文案
clawhub install copywriting

# SEO审计
clawhub install seo-audit
```

---

## 💰 金融相关

```bash
# 股票行情
clawhub install alphaear-stock

# 市场预测
clawhub install alphaear-predictor

# 情绪分析
clawhub install alphaear-sentiment
```

---

## 🛠️ 开发必备

```bash
# GitHub自动化
clawhub install github

# Docker管理
clawhub install docker-essentials

# 代码审查
clawhub install pr-reviewer
```

## 📊 数据采集

```bash

```

## 📅 生产力


## 🔍 快速搜索

```bash
# 搜索Skills
clawhub search <关键词>

# 本地查找
./skills_library/find_skill.sh <关键词>
```
